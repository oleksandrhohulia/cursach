Перевірка коректності

Перевірка коректності введених даних у форму перед відправкою: Перш за все, необхідно перехопити подію відправки форми за допомогою JavaScript. Для цього можна використовувати обробник події "submit". Після цього можна здійснювати перевірку коректності даних. Наприклад:

var form = document.getElementById("myForm"); form.addEventListener ("submit", function(event) { // Перевірка коректності даних if (!validateForm()) { event.preventDefault(); // Відміна відправки форми } }); function validateForm() { // Код для перевірки даних у формі // Повертає true, якщо дані коректні, або false в іншому випадку }

Перевірка
обов'язкових полів

Для перевірки обов'язкових полів можна перевіряти, чи є введені значення відповідних полів пустими. Для перевірки формату даних, наприклад, електронної пошти або номера телефону, можна використовувати регулярні вирази. Наприклад:

function validateForm() { var nameInput = document.getElementById ("name"); var emailInput = document.getElementById ("email"); if (nameInput.value === "") { alert ("Будь ласка, введіть своє ім'я"); return false; } var emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/; if (!emailRegex.test(emailInput.value)) { alert ("Будь ласка, введіть коректну адресу електронної пошти"); return false; } return true; }

Відображення
повідомлень
про помилки

При виявленні помилок під час валідації форми, можна відобразити відповідні повідомлення для користувача. Наприклад, можна використовувати HTML-елементи для відображення повідомлень або додати спеціальний блок повідомлень під формою. Наприклад:

function validateForm() { var nameInput = document.getElementById("name"); var nameError = document.getElementById("nameError"); if (nameInput.value === "") { nameError.textContent = "Будь ласка, введіть своє ім'я"; return false; } else { nameError.textContent = ""; } // Інші перевірки та повідомлення про помилки return true; }

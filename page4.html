Cпливаючі вікна та модальні вікна - це елементи веб-сайту, які з'являються на екрані користувача поверх основного вмісту сторінки. Вони використовуються для показу додаткової інформації, повідомлень, форм або будь-якого іншого контенту, який потребує особливої уваги користувача. Використання JavaScript для відкриття та закриття спливаючих вікон: Для відкриття спливаючих вікон з використанням JavaScript, можна використовувати функцію window.open(). Ця функція приймає параметри, такі як URL-адреса, розмір вікна, налаштування тощо. Наприклад:

function openPopupWindow() { var url = "popup.html"; var name = "popupWindow"; var width = 500; var height = 300; var left = (window.innerWidth - width) / 2; var top = (window.innerHeight - height) / 2; var features = "width=" + width + ", height=" + height + ", left=" + left + ", top=" + top; window.open (url, name, features); }

Для закриття спливаючих вікон можна використовувати функцію window.close(). Наприклад:

function closePopupWindow() { window.close(); }

Додавання затемнення заднього фону при відображенні модальних вікон: Для створення модальних вікон з затемненням заднього фону, можна застосувати додатковий елемент, який перекриватиме основний контент. Цей елемент може мати фоновий колір з прозорістю або затемнюватися за допомогою CSS-стилів. Наприклад:

function openModalWindow() { var modal = document.getElementById ("modal"); modal.style.display = "block"; } function closeModalWindow() { var modal = document.getElementById ("modal"); modal.style.display = "none"; }

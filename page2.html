Введення до JavaScript:
змінні, функції, умови

Змінні:
Змінні в JavaScript використовуються для збереження інформації. Вони можуть містити числа, рядки, булеві значення та інші типи даних. Для оголошення змінної використовується ключове слово "var", "let" або "const". Наприклад:

var age = 25; let name = "John"; const PI = 3.14;

Функції:
Функції в JavaScript використовуються для організації блоків коду, які можуть бути викликані для виконання певних завдань. Функції можуть приймати параметри і повертати значення. Наприклад:

function add(a, b) { return a + b; } var result = add(2, 3); // result = 5

Умови:
Умови в JavaScript використовуються для виконання певних дій в залежності від заданих умов. Найпоширенішими конструкціями умов є "if", "else if" та "else". Наприклад:

var age = 18; if (age <18) { console.log("Ви ще неповнолітній"); } else if (age <=18 && age <65) { console.log("Ви дорослий"); } else { console.log("Ви пенсіонер"); }

Маніпулювання DOM
(Document Object Model)
з використанням JavaScript

DOM - це структура, яка представляє HTML-документ у вигляді дерева об'єктів. JavaScript надає можливість маніпулювати цим деревом, змінювати елементи, їх властивості тастилі. Для доступу до елементів DOM використовуються методи та властивості JavaScript. Наприклад:

// Отримання посилання на елемент за його ідентифікатором var element = document.getElementById("myElement"); // Зміна текстового вмісту елемента element.textContent = "Привіт, світ!"; // Зміна стилю елемента element.style.color = "red";

Реагування на події користувача
з допомогою JavaScript

JavaScript дозволяє реагувати на події, які виникають під час взаємодії користувача з веб-сайтом, такі як натискання кнопок, введення тексту, переміщення миші тощо. Для обробки подій використовуються обробники подій (event handlers) JavaScript. Наприклад:

// Отримання посилання на кнопку var button = document.getElementById("myButton"); // Додавання обробника події для натискання кнопки button.addEventListener("click", function() { console.log("Кнопку натиснуто!"); });

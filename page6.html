Додавання інтерактивності до кнопки за допомогою JavaScript: За допомогою JavaScript можна додати реакцію на події, що відбуваються з кнопкою, такі як натискання, наведення курсора, відпускання тощо. Для цього можна використовувати методи addEventListener() та removeEventListener(). Наприклад:

var button = document.getElementById("myButton"); button.addEventListener("click", function() { // Виконати певні дії при натисканні на кнопку console.log("Кнопка була натиснута!"); });

міна стилів кнопки при наведенні курсора миші: Для зміни стилів кнопки при наведенні курсора миші можна використовувати події mouseover та mouseout. Наприклад:

button.addEventListener ("mouseover", function() { // Змінити стилі кнопки при наведенні button.style.backgroundColor = "red"; button.style.color = "black"; }); button.addEventListener ("mouseout", function() { // Повернути початкові стилі кнопки при знятті курсора button.style.backgroundColor = "blue"; button.style.color = "white"; });

Цей код змінить колір фону та тексту кнопки при наведенні курсора на неї, а потім поверне початкові стилі при знятті курсора. Це додасть інтерактивність до кнопки, показуючи користувачеві візуальний зворотний зв'язок.
